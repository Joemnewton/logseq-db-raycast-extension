# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Quick Capture command to add notes to today's journal
- Search Graph command with dual mode support (API and local)
- Add Task command with status, priority, and deadline properties
- Support for Logseq database graphs (v2.0.0+)
- Integration with Logseq CLI (@logseq/cli)
- MCP Server integration for quick capture and task creation
- Optional timestamp prepending for captured notes
- Optional tag addition for quick captures
- Local search mode for offline usage
- Comprehensive error handling and user feedback
- Full TypeScript implementation
- Documentation: README.md, SETUP.md, and CONTRIBUTING.md

[1.0.0]: https://github.com/joenewton/logseq-db-raycast-extension/releases/tag/v1.0.0
